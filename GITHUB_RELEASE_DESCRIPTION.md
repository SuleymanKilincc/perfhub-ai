## 🎉 PerfHub AI v4.0.0 - Major Release

AI-powered hardware analysis with Gemini 2.5 Flash, Ray Tracing/Path Tracing support, and enhanced FPS prediction.

## ✨ What's New

### 🤖 AI Assistant (Gemini 2.5 Flash)
- Embedded API key - no setup required
- Hardware analysis & recommendations
- Turkish & English support

### 🌟 Ray Tracing / Path Tracing
- 32 games with RT/PT support
- Toggle RT/PT per game
- Realistic performance impact (-40% RT, -65% PT)

### 💾 RAM Selector & Impact
- Selectable RAM: 4GB - 128GB
- Game-specific RAM sensitivity
- Realistic FPS penalties for low RAM

### 🖥️ Intel iGPU Support
- 6 Intel integrated GPUs added
- Proper scoring for iGPUs
- Better laptop detection

### 🌐 Global Language Selector
- 🇹🇷 Türkçe / 🇬🇧 English
- Sidebar language picker
- Affects all AI responses

### 🎨 Application Icon
- Custom PerfHub AI logo
- Professional branding
